Obfuscated Collaboration
-Unique views of the same document for each user depending upon their clearance levels.
-
-Provides safe collaboration without duplication.
A perfectly legitimate case could be where we want different views for clients according to their level. So we can provide an obfuscated view of the data to the user. However the user has no idea about that and he can continue working on that view. As everything is saved in the form of computations this gives us the ability to incorporate the changes made by the user on the obfuscated view directly on the original data in a collaborative setting. This also eliminates the need of creating multiple files.
